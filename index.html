Main File Structure

All the file are well organized, its so easy to work with the template.
1. Unzip the main-files.
2. Open "MainFile" folder.
You will find the twelve html files.
Also you will see into "assets" 5 directories, css, fonts, images, js, and sass.
In the first folder Content you will find style.css and other styling files the all the other files expect the htmls.
Js folder contain only the JavaScript library. Fonts folder contain essetial font files for fonts.
